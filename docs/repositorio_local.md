# Cómo crear un repositorio local en GitBash

Para crear un repositorio local es importante entender los comandos básicos de registro los cuales son:

1. git config --global user.name
Este comando sirve para registrar el nombre de usuario.
2. git config --global user.email
Este comando sirve para registrar el email del usuario.
3. git init
Este comando inicializa un repositorio.
4. Git add .
Añade todas las cosas sin guardar a la rama estando listas para ser commited.
5. Git commit -m "mensaje"
Sirve para guardar los cambios hechos en el repositorio local.
